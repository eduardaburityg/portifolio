# Meu Portfólio

Este é meu portfólio pessoal feito com HTML, CSS e animações do Font Awesome. Com estrutura e templates do [Undraw](https://undraw.co/illustrations/5).

## ✨ Tecnologias Utilizadas

- HTML5  
- CSS3  
- Font Awesome (com animações como `fa-beat`)  
- Estrutura responsiva (usando classes como `col-md`, `col-sm`...)

## Objetivo

Criar um portifólio pessoal simples que apresente minhas habilidades, ferramentas e contato.  
Adicionei ícones personalizados com algumas animações animações sobre as seções, e inserindo um botão direcionando à um jogo.
